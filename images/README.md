# Images Source Map

Images are the visual source for Rich Bizness Universe.

Use real files from this folder before choosing placeholders.

Main lanes:

- brand assets
- home portal assets
- profile photos and banners
- avatar and character assets
- meta world assets
- music podcast radio covers
- live and watch thumbnails
- store product images
- sports covers
- gaming covers

Rule: every redesign should check image assets first, then wire the best real asset into the page or feature owner.
